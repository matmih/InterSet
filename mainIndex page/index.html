Redescription mining [5] is a field of Knowledge Discovery that aims to find alternative characterizations of similar element subsets. In addition, it allows users to find connections between attributes from different attributes sets, called views. This is useful in many different fields of science like biology, economy, medicine etc.


The main purpose of the tool INTERSET is to allow interactive, comprehensive, redescription set exploration. On this page you can test the features of the tool by exploring redescriptions created on data describing world countries by using general country information and country trading patterns for the year 2012 ([2,6,7]). The tool is described in more detail in the paper "Interactive redescription set exploration" submited to the special issue of the ACM Transactions on Knowledge Discovery from Data on the topic of Interactive Data Exploration and Analytics (2015.)


General data information

We use data describing 199 world countries. The data contains two views:

  • Country trade information: contains 312 numerical attributes

  • General country information: contains 49 numerical attributes

On this data, we created 4150 redescriptions by using a Predictive Clustering Trees [3] based algorithm for mining redescriptions. The main idea of redescription generation process can be seen in [4].


Redescription set exploration

The tool offers interactive redescription set exploration based on elements described by redescriptions in the redescription set, attributes contained in the redescription queries and general redescription properties. The properties used in the literature are: the Jaccard index, the p-value and redescription support [1].
