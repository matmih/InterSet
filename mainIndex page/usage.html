How to use the tool

The tool consists of three different redescription set exploration views that can be selected at the tools main page :

  • Element based redescription set exploration
  • Attribute based redescription set exploration
  • Redescription set exploration based on redescription quality measures


The main steps on how to use each of this exploration views are explained on the corresponding view page. On each user click in the interface, a new component is displayed below currently displayed components. Please note, that you might need to scroll to see all the components.

Limitations

The tool is optimised to work on Google Chrome browser . It also works on other more popular browsers like Mozilla Firefox and Internet Explorer . However, depending on the number of redescriptions and the number of measures used, some lag in crossfilter performance can be detected even when Google Chrome works without problems.

Currently, the tool outputs redescription support as a html table, this will be replaced with output to file option for supports lager than 100000 elements.
